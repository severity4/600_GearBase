# 映奧創意工作室 — 器材管理 Web App 部署指南

## 架構說明

這個 Web App 使用 **Google Apps Script** 搭配 **Google Sheets** 作為後端資料庫。

- **員工後台**：完整的器材、租借、客戶、庫存、盤點、付款管理
- **客戶前台**：瀏覽器材目錄、加入購物車、送出租借申請

---

## 部署步驟

### 步驟 1：準備 Google Sheets

確保你的 Google Sheets 已經有以下 21 個分頁（如果之前用了 Excel 範本，應該已經有了）：

Storage_Locations、Equipment_Types、Equipment_Units、Accessory_Bindings、
Maintenance_Logs、Customers、Staff、Rentals、Rental_Items、Service_Items、
Rental_Addendums、Discount_Rules、Overdue_Rules、Damage_Records、
Wear_Tolerance、Credit_Notes、Payments、Inventory_Logs、Print_Templates、
Stocktake_Plans、Stocktake_Results

### 步驟 2：建立 Apps Script 專案

1. 開啟你的 Google Sheets
2. 選單 → **擴充功能** → **Apps Script**
3. 這會開啟 Apps Script 編輯器

### 步驟 3：貼入程式碼

在 Apps Script 編輯器中：

1. **Code.gs**（主程式）
   - 把預設的 `Code.gs` 內容全部刪除
   - 貼入 `Code.gs` 的內容

2. **API.gs**（資料存取層）
   - 點擊左側 `+` → **指令碼** → 命名為 `API`
   - 貼入 `API.gs` 的內容

3. **StaffApp.html**（員工後台介面）
   - 點擊左側 `+` → **HTML** → 命名為 `StaffApp`
   - 貼入 `StaffApp.html` 的內容

4. **CustomerApp.html**（客戶前台介面）
   - 點擊左側 `+` → **HTML** → 命名為 `CustomerApp`
   - 貼入 `CustomerApp.html` 的內容

5. **Styles.html**（共用樣式）
   - 點擊左側 `+` → **HTML** → 命名為 `Styles`
   - 貼入 `Styles.html` 的內容

6. **JavaScript.html**（共用 JavaScript）
   - 點擊左側 `+` → **HTML** → 命名為 `JavaScript`
   - 貼入 `JavaScript.html` 的內容

### 步驟 4：部署 Web App

1. 在 Apps Script 編輯器中，點擊右上角 **部署** → **新增部署作業**
2. 類型選擇 **網頁應用程式**
3. 設定：
   - **說明**：器材管理 Web App v1.0
   - **執行身分**：我（你的 Google 帳號）
   - **存取權**：
     - 如果只讓公司內部用：「只有自己」或「貴機構的所有使用者」
     - 如果要讓客戶也能使用：「所有人」
4. 點擊 **部署**
5. 複製產生的 **Web App URL**

### 步驟 5：存取 Web App

部署完成後你會得到一個 URL，格式類似：
```
https://script.google.com/macros/s/xxxxx/exec
```

- **員工後台**：直接開啟 URL（預設就是員工後台）
- **客戶前台**：URL 加上 `?mode=customer`，例如：
  ```
  https://script.google.com/macros/s/xxxxx/exec?mode=customer
  ```

## 使用說明

### 員工後台功能

| 頁面 | 功能 |
|------|------|
| 儀表板 | 器材總覽統計、租借狀態、營收概況 |
| 器材管理 | 新增/編輯器材類型與個體、自動產生內部編號 |
| 租借管理 | 建立/查看/更新租借單、狀態追蹤 |
| 客戶管理 | 客戶資料 CRUD、搜尋篩選 |
| 庫存管理 | 存放位置管理、出入庫紀錄 |
| 盤點 | 建立盤點計畫、執行盤點 |
| 付款 | 付款紀錄管理 |
| 報表 | 資料完整性檢查、庫存差異報告 |

### 客戶前台功能

1. 客戶瀏覽器材目錄（只顯示 `available` 的器材）
2. 用分類按鈕篩選器材
3. 點擊「加入清單」放入購物車
4. 點擊「結帳」填寫個人資訊和租借日期
5. 送出後系統自動建立 draft 狀態的租借單
6. 員工在後台確認並處理

## 更新部署

修改程式碼後：
1. Apps Script 編輯器 → **部署** → **管理部署作業**
2. 點擊鉛筆圖示（編輯）
3. 版本選擇 **新版本**
4. 點擊 **部署**

---

## 注意事項

- Google Apps Script Web App 有每日執行額度限制（免費帳號約 6 分鐘/次、90 分鐘/天）
- 大量資料時建議分頁載入
- 客戶前台 URL 可以設定為公司網站的連結或 QR Code
- 建議定期備份 Google Sheets 資料
